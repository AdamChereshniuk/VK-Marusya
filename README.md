## Проект: VK Marusya 🎬

**VK Marusya** — это веб-приложение для поиска и оценки фильмов.


🌐 Просмотр:
[VK Marusya](https://adamchereshniuk.github.io/VK-Marusya/)

🎨 Макет:
[Figma](https://www.figma.com/design/8FW6Yt3ztcoYATQhqiy4qK/Макет-VK-Маруся-для-диплома-Vue-React?node-id=0-1&p=f&t=E9UZGoCm9UOtCMnQ-0)

### 🚀 Установка:

Установка зависимостей:
```
npm install
```
Запуск сервера разработки:
```
npm run dev
```
Запуск production сборки:
```
npm run build
```

### 🛠️ Технологии:

- [React](https://react.dev/)
- [Redux](https://redux-toolkit.js.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
